# StudyJNS 📚

Free entry test preparation platform for Pakistani students targeting MUET, DUET, and University of Sindh.

## Features
- 📝 MCQ Quizzes by topic
- 📊 Personal progress tracking
- 🏆 Leaderboard
- 📅 3-Month study plan
- 📄 Past papers & free resources

## Built With
- HTML · CSS · JavaScript
- Firebase Auth + Firestore
